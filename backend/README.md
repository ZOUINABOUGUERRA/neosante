# NéoSanté - Backend Firebase Configuration

## Architecture Overview

This backend powers the NéoSanté Intelligent Neonatal System with:

- **Firestore** - Primary database for all medical records
- **Firebase Storage** - Secure storage for medical images and documents
- **Cloud Functions** - Serverless backend for AI, notifications, backups, and PDF generation
- **Firebase Authentication** - Secure user management with role-based access
